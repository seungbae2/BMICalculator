### BMI 계산하기

공식 =  몸무게(kg) / (키 (m) * 키 (m)) 

18.5 이하면 저체중

18.5 ~ 23 은 정상체중

23 ~ 25는 과체중

25 ~ 30 은 경도 비만

30 ~ 35 는 중정도비만

35 이상은 고도 비만
